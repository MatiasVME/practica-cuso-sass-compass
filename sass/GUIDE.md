
# Guide 

El CSS debe seguir los patrones de OOCSS y SMACSS. 

## Convenciones

### Comentarios

```
//
// Primer nivel
//
 
// Segundo nivel 
// 

// Tercer Nivel

$foo: 10; // Tercer nivel (igual)
```

### Nombre para los módulos, layouts y otros

btn-*			# botón
blk-*			# bloque
l-* 			# layout
clr				# color

### Nombre para los estados

is-on, is-of
is-active / is-selected
is-pressed
is-hidden
is-animated
is-disabled
is-collapsed
